# Color
Простой интерактивный проект, где при клике на цветной кружок заголовок меняет цвет в соответствии с цветом кружка.

## Демонстрация

На странице расположены разноцветные кружки. При клике на любой из них цвет заголовка меняется на цвет выбранного кружка.

## Функционал

- 8 цветных кружка для взаимодействия
- При клике на кружок заголовок меняет цвет
- Плавная анимация изменения цвета
